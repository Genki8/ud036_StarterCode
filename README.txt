# How to run the code

To use this project you must have python installed
Program was designed using python version 2.7

Steps to run this file
1. Download the code of clone the repository from https://github.com/Genki8/ud036_StarterCode
2. Unzip the files to the location of your choosing
3. Open a command window and navigate to the location you created in step 2.
4. Run : $entertainment_center.py which will generate the html and open it in a browser.
5. Enjoy!
